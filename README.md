# Sistema de Gestión Hospitalaria

Aplicación web full-stack para gestionar pacientes, traslados entre áreas, historial médico, notas clínicas y equipos médicos con una interfaz moderna y completamente responsive.

## Qué incluye
- Login y registro por roles
- Dashboard clínico y operativo
- Gestión de pacientes con flujo por pisos y áreas
- Perfil completo del paciente con tabs: Sobre, Historial, Movimientos, Pruebas/Imágenes y Notas
- Gestión de equipos con alta, mantenimiento y asignación
- Diseño pensado para móvil y escritorio

## Estructura
- backend/: API Express con autenticación, roles y CRUD
- frontend/: interfaz web con Tailwind CSS y JavaScript vanilla
- database/: esquema SQL y datos de ejemplo

## Requisitos
- Node.js 18+
- npm

## Instalación
```bash
cd /workspaces/proyect-gestos-hospitalario
npm install
```

## Ejecución
```bash
npm start
```

Abrir en el navegador:
```text
http://localhost:3000
```
